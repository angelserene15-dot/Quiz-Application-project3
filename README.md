**Quiz Application in C**

**Project Overview:**

The Quiz Application is a console-based program developed in C that allows users to answer multiple-choice questions and receive an instant score based on their performance. The application presents a series of questions, evaluates user responses, calculates the final score, percentage, and grade, and displays the results at the end of the quiz.

This project demonstrates the implementation of structures, arrays, strings, loops, and conditional statements in C while creating an interactive and educational application.


**Objectives:**

Develop a menu-driven quiz system using C.

Evaluate user knowledge through multiple-choice questions.

Calculate scores and percentages automatically.

Implement data storage using structures.

Enhance programming skills through practical application development.


**Features:**
